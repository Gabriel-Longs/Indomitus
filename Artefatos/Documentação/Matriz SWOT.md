# Matriz SWOT

## Análise de Ambiente

| Forças (Strengths) | Fraquezas (Weaknesses) |
| :--- | :--- |
| - Personalização por demanda<br>- Treinos com embasamento científico<br>- Ciclos de feedback contínuos<br>- Ecossistema de saúde integrado<br>- Serviço de IA local próprio | - Custo de manutenção alto para o idealizador<br>- Alto processamento de dados por IA exigindo infraestrutura<br>- Manipulação de dados sensíveis de saúde<br>- Necessidade de hardware de ponta |

| Oportunidades (Opportunities) | Ameaças (Threats) |
| :--- | :--- |
| - Fornecer dados fisiológicos com alta precisão<br>- Integração futura com profissionais da área de saúde<br>- Parcerias com marcas da área do desenvolvimento físico<br>- Gamificação para maior engajamento | - Alta competitividade com grandes apps (Hevy, Runna)<br>- Nicho com possível engajamento inicial baixo<br>- Regulação de termos da classe profissional de saúde<br>- Risco de "alucinação" do modelo de IA |
