# Matriz 5W2H

## Planejamento Estratégico

| Dimensão | Pergunta | Resposta |
| :--- | :--- | :--- |
| **What** | O QUÊ será feito? | Desenvolver um app mobile para criação e acompanhamento de treinos baseados em IA. |
| **Why** | POR QUÊ será feito? | Para barrar a falta de personalização direta em treinos para os usuários, tornando a saúde acessível. |
| **Where** | ONDE será utilizado? | Em celulares (aplicativo mobile) e um painel administrativo web para gestão. |
| **When** | QUANDO deve ser entregue? | Lançamento do MVP planejado para o prazo de 1 ano. |
| **Who** | QUEM são os responsáveis? | A dupla de fundadores/idealizadores (Desenvolvimento, DevOps e Estratégia). |
| **How** | COMO será feito? | Stack tecnológica: TypeScript, React, PostgreSQL. Desenvolvimento em modelo contínuo (tempo livre). |
| **How Much** | QUANTO vai custar? | Financiamento via recursos próprios (bootstrapping) da dupla, com teto de R$ 3.000 focados na infraestrutura/IA local. |
